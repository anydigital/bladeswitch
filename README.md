![GitHub Release](https://img.shields.io/github/v/release/anydigital/bladeswitch?color=black&label=%20)

The only starter compatible with _BOTH_ (!) Jekyll and Eleventy at the same time.

It allows you to develop locally without [Ruby friction](https://any.digital/tricks/ruby/) (thanks to 11ty), and deploy natively to GitHub Pages using built-in Jekyll engine ‚Äî no GitHub Actions required!

Now, also includes [Sveltia CMS](https://github.com/sveltia/sveltia-cms). All of this ‚Äî with only <big>3</big> direct dependencies:
